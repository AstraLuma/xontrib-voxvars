VoxEnv
===============================

version number: 0.0.1
author: Jamie Bliss

Overview
--------

Tie envvars to your vox envs

Installation / Usage
--------------------

To install use pip:

    $ xpip install xontrib-xontrib-voxenv


Or clone the repo:

    $ git clone https://github.com/astronouth7303/xontrib-voxenv.git
    $ xpip install ./xontrib-voxenv
